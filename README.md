# WebGIS Road Map (QGIS + Leaflet)

This project is a simple WebGIS map created using **QGIS**, **GeoJSON**, and **Leaflet.js**.
It visualizes road infrastructure layers exported from QGIS and displayed on a web map.

## Features

* Interactive web map
* Road layers styled by type
* Popup information for each feature
* OpenStreetMap basemap
* GeoJSON data integration

## Technologies

* HTML
* JavaScript
* Leaflet.js
* QGIS
* GeoJSON
* OpenStreetMap

## Project Structure

```
.
├── index.html
├── merge.geojson
└── mergepol.geojson
```

## Data Preparation (QGIS)

1. Prepare vector layers in QGIS
2. Merge layers if needed
3. Export layers as **GeoJSON**
4. Use the GeoJSON files in the Leaflet web map

## How to Run

Simply open `index.html` in a browser or deploy using **GitHub Pages**.
